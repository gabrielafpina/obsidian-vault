#Obsidian/regex

**New note**
cmd + N

**Link creation**
Cmd + K

**Zoom in**
Cmd +

**Zoom out**
Cmd -

**Link to other pages**
[[

**Open search**
Cmd + o

**Creation of other pages**
Higlhight what you want + clicar 

**Display content**
Add ! before link
Eg.:
![[heading t#^Heading testing]]


**Display heading**
Add # to the end of the link and select heading
![[heading t#Heading testing]]

**Display content**
Add ˆ to the end of the link and select content. This is good to reference small parts of texts within other texts.
![[heading t#^aac5f0]]

**Access search**
Cmd + Shift + F

**Creating a callout**
> [!info]

>[!warning]

>[!note]

>[!important]
