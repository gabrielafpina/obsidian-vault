###### **DEFINIÇÃO:**
Os trilhos estão relacionados ao conceito de pausa na imagem. [[O DHS gera uma imagem em pausa]]. Ou seja, os fatores que estão relacionados àquele momento do DHS, o cérebro memoriza - e para  [[A percepção é sempre pessoal]]. São todos os alérgenos - tudo que conhecemos como alergias (alimentos, sabores, cheiros, pessoas - algumas pessoas são alérgicas a outras, porque trazem uma memória de separação) e intolerâncias, são trilhos do conflito.

Para cada pessoa, os trilhos do conflito são os 5 sentidos, o tempo (datas específicas...), pessoas envolvidas, o espaço, memorizando os fatores envolvidos em um DHS. [[O que é um DHS?]] A sua função é servir como um alerta para evitar uma experiência 'perigosa' uma segunda vez.

É sempre a memória de algo com uma temática de separação envolvida. [[Tudo para o cérebro é uma noção de sobrevivência]]. 

Para resolver um trilho, o caminho é entender o que, para aquela pessoa, representa esse perigo. Os trilhos nos ajudam a compreender de onde vem o conflito. 

>[!important] Para começar a tratar os trilhos do conflito com o paciente, perguntar: em que circunstâncias o sintoma acontece? 

Vem mais de manhã, mais de noite, a alergia é a algum alimento específico, ou a alguma situação específica?

OBS: algumas pessoas chamam os trilhos de conflito de 'gatilhos'.
