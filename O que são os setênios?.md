###### **DEFINIÇÃO:**
Afirma que a vida muda radicalmente a cada 7 anos: 7, 14, 21, 28...

[[Rudolf Steiner]] dá muita importância e prioridade para os 3 primeiros setênios, que é onde o Ser está se preparando para a vida.

Quem tiver uma base sólida nesses 3 primeiros setênios terá harmonia e tranquilidade nos demais.

Steiner afirma que existem determinadas questões que não podemos deixar de vivenciar nos setênios.
