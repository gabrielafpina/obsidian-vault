###### **DEFINIÇÃO:**
"O inconsciente permite bloquear o pensamento de qualquer coisa desagradável. Isso contribui para proteger o ser humano." (O poder do inconsciente)

"Certas proteínas reguladoras formam uma bainha em torno do DNA, o que impede a leitura dos genes. Elas são uma espécie de proteção das informações primordiais." - [[Bruce Lipton]]

Quando o paciente está fora do conflito, dizemos que ele está mini-esquizofrênico. Mas para sua doença, ele é 100% esquizofrênico, ou seja, maxi-esquizofrênico.

De maneira conceitual, um paciente esquizofrênico é aquele que vive em uma outra realidade, acreditando nessa outra realidade que é distorcida. Na verdade, *todos nós somos esquizofrênicos para os nossos conflitos*. 
	Ex: conselho que você dá para a pessoa, mas que quando você está com um desafio, você mesmo não segue o conselho.

Quando vivemos o DHS ([[O que é um DHS?]]), a memória daquele trauma fica encapsulada com o objetivo de nos protegermos daquela memória. Se lembrarmos da situação, para o cérebro, é como se ainda estivéssemos vivendo aquela situação e isso esgota a fisiologia dela. [[Tudo para o cérebro é uma noção de sobrevivência]].

Quando você vive medos muito fortes, você passa de uma realidade consciente para uma realidade imaginária. É um modo de proteção.

A maior expressão de minimax é a **negação** (recusa psicológica). É o 'não, mas...'.
